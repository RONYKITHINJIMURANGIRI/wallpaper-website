# VersaceHub

A modern Next.js landing experience for a free gaming wallpaper gallery.

## What’s included

- Next.js 16 App Router site
- Tailwind CSS v4 styling
- client-side navigation and search
- fullscreen lightbox preview
- responsive homepage and detail sections
- launch-ready GitHub repository setup

## Scripts

```bash
npm install
npm run dev
npm run build
npm start
```

## Notes

- The site is implemented as a client-driven page in `app/page.jsx`.
- Static build artifacts are ignored by `.gitignore`.
- Use `npm run dev` to open the site locally at `http://localhost:3000`.
